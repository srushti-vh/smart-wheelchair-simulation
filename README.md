# AI-Powered Smart Wheelchair for Disabled Individuals with Voice Control

## Project Overview
This project proposes an AI-powered smart wheelchair system that enables disabled individuals to control wheelchair movement using voice commands. The system focuses on improving mobility, independence, and accessibility through speech recognition and intelligent control mechanisms.

## Features
- Voice-controlled navigation
- AI-based command recognition
- Hands-free wheelchair operation
- Improved accessibility for disabled users
- Simulation-based implementation

## Technologies Used
- Python
- Speech Recognition
- Artificial Intelligence
- VS Code
- GitHub

## Documents Included

- System Design
- Final Project Report

## Project Objectives
- Enable wheelchair movement through voice commands
- Improve user independence
- Provide a cost-effective smart mobility solution
